# Checklist

## General streaming

- Start OBS
  - Ensure VS Code and Firefox are correctly selected in scenes
  - Turn off auto switcher
  - Pick countdown scene
- Open restream
  - Set stream title
- Start stream in OBS
- Wait for stream countdown
- Start recording in OBS
